# 夢見る少女は眠れない

起こさずに、待てますか。ねむりを ON/OFF ではなく 0.0〜1.0 の Float で愛でる、**二重放置ゲーム**。

**Live: https://osakenpiro.github.io/yumemi-shoujo/**

- ① アプリを**ひらいていない時間**が、夢を濃くする（ゆめのこさ Float）
- ② ひらいても、少女は寝ている。**自然に起きるまで待つ**のがいちばんの遊びかた（ねむり Float 1.00 → 0.00）
- タップ連打で起こすこともできる。でも夢は失われる（罪悪感設計）
- 目ざめた少女は夢を語る。その夢は「あなたが留守中にしていたこと」（自己申告・案A）で 1.4x だけ偏る——気のせいかもしれないくらい、すこしだけ

画風『ひとあかり』／ 設計思想 Boolean → Float (BeeF) ／ 単一HTML + localStorage + GitHub Pages（ゼロコスト・ゼロメンテ）

roadmap: 嘘を見抜く少女（信頼度Float・案C）／コウノトリ／天気連動の夢
